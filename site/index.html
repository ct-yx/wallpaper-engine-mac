BUILT FOR YOUR DESKTOP

不只是一张静态图片

把 Wallpaper Engine 创意工坊里的优秀作品，带到 macOS 的每一个桌面空间。

01

场景、视频与网页壁纸

支持视频文件、HTML/WebGL 网页壁纸和静态场景图层；无法完整渲染的场景会优雅回退到预览图。

02

多显示器 · 多桌面

不同屏幕设置不同壁纸，切换 macOS Spaces 时持续播放，不打断你的沉浸感。

✦

03

内置创意工坊浏览

搜索、筛选、排序和下载一气呵成。也支持本地文件夹与 ZIP 壁纸包导入。

04

按你的节奏播放

状态栏快速切换最近壁纸；其他应用获得焦点时可暂停、静音或停止播放，兼顾桌面效果与性能。

GET STARTED

三步，让桌面动起来。

发布包由 GitHub Actions 自动构建并以 ad-hoc 方式签名，不受 Apple Personal Team 七天描述文件期限影响。

下载 v0.8.1 查看校验文件与全部版本 ↗
  1. 01
    下载并解压

    从 GitHub Release 下载最新 ZIP 包，旁边提供 SHA-256 校验文件。

  2. 02
    将应用移到“应用程序”

    首次打开若遇到系统提示，请在“隐私与安全性”中确认打开。

  3. 03
    连接你的壁纸库

    使用内置 Workshop，或直接导入已下载的文件夹和 ZIP 壁纸包。

关于首次启动：为避免过期的个人开发者证书，Release 使用 ad-hoc 签名，尚未经过 Apple 公证。macOS 可能在首次打开时显示安全确认；这不影响应用正常使用。
